This document contains the optimized Title Tags and Meta Descriptions for the homepage and all collection pages. 

**Instructions for Shopify:**
1. Go to Shopify Admin > Online Store > Preferences (for Homepage).
2. Go to Shopify Admin > Products > Collections (for Collection pages).
3. Scroll down to the "Search engine listing preview" section at the bottom of the page.
4. Click "Edit website SEO" and paste the content below.
