## 一、周度执行总览

每周固定投入时间：**约 3-4 小时**

| 模块 | 时间 | 频率 |
|------|------|------|
| 社媒内容发布 | 60 分钟 | 每周 |
| SEO 内容更新 | 60 分钟 | 每周 |
| 外链建设 | 30 分钟 | 每周 |
| 数据复盘 | 30 分钟 | 每周 |
| 站内优化 | 30 分钟 | 按需 |

## 二、每日社媒发布 Checklist

### Twitter/X（每日 1-2 条）

- [ ] 早上 9:00 EST 发布产品图推文（使用模板库）
- [ ] 下午 3:00 EST 发布互动内容（问题/投票/转发）
- [ ] 回复所有 @mentions 和评论
- [ ] 关注并互动 3-5 个键盘圈账号

### TikTok（每周 3-4 个视频）

- [ ] 周一：产品特写视频（15-30秒）
- [ ] 周三：制作过程/工作室 Vlog
- [ ] 周五：Before/After 桌面对比
- [ ] 周日：开箱/包装展示

### Reddit（每周 2-3 帖）

- [ ] 周二：在 r/ArtisanKeycaps 或 r/MechanicalKeyboards 发帖
- [ ] 周四：在 r/DeskSetup 分享桌面图
- [ ] 持续：回复相关帖子评论

### Pinterest（每周 5-10 个 Pin）

- [ ] 将本周新产品图上传至对应看板
- [ ] 为每个 Pin 添加完整描述和链接
- [ ] 检查是否有高流量 Pin 需要优化

## 三、每周 SEO 任务 Checklist

### 内容发布

- [ ] 发布/更新 1 篇博客文章（按内容日历）
- [ ] 检查文章是否包含目标关键词（标题、H2、正文）
- [ ] 添加 2-3 个产品内链
- [ ] 添加 1-2 个相关文章内链
- [ ] 文章末尾添加 CTA 按钮

### 技术 SEO

- [ ] 检查本周新增产品页的 meta description
- [ ] 检查新上传图片的 ALT 标签
- [ ] 运行 `python3 06-scripts/seo_audit.py` 生成周报

### 关键词监控

- [ ] 在 Google Search Console 检查排名变化
- [ ] 记录新进入前 50 名的关键词
- [ ] 更新关键词追踪表

## 六、内容素材复用规则

### 一张产品图 → 多平台内容

```
1张高质量产品图
  ├── Twitter：1200×675 横版 + 文案
  ├── Instagram：1080×1080 正方形 + 故事文案
  ├── Pinterest：1000×1500 竖版 + 详细描述
  ├── TikTok：截取为视频封面
  └── Reddit：原图 + 制作故事文案
```

使用 `06-scripts/social_cover_generator.py` 批量生成各平台尺寸。

## 八、紧急情况处理

### 负面评论/差评

1. 24小时内回复
2. 私信道歉并提供解决方案（退款/补发）
3. 不删除评论（除非违规）
4. 公开回复展示品牌态度

### 产品缺货

1. 立即在产品页添加"Notify me"按钮
2. 在社媒发布"即将补货"预告
3. 向邮件订阅者发送补货通知

### 爆款内容

1. 立即在其他平台转发/复制
2. 增加该产品的广告预算（如有）
3. 确保产品库存充足
